# Contributing

* filename format should be `nvidia-<version>-dkms`
* the file should contain the table in the following format

```markdown
| NVE0 | NVE4 | GK104 | Description |
|------|------|-------|-------------|
| NV50 | NV50 | G80 | GeForce 8800 (GTS, GTX, Ultra), Quadro FX (4600 (SDI), 5600) |
```

* Generate the NVIDIA-TABLE.md by ` python ./gen_table.py`.
